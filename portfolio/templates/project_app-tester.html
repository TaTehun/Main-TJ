App Installation Tester is a fully automated GUI-based tool to install and test Android apps across multiple devices simultaneously.

It supports Google Play-based installation, automatic launch, dark/light mode toggling, multi-window testing, crash detection (via logcat), monkey UI interaction, and CSV-based tracking.

Users can import and edit test app lists via a table editor, execute parallel tests, and export the full result report.

Technologies used

  • Python
  • PyQt5
  • pandas
  • uiautomator2
  • ADB
