The SocialMedia application is a full-featured social media platform currently under development, with an estimated completion date in May 2024.

The application aims to offer a rich user experience, allowing users to create posts, follow other users, and interact through comments and likes.

Key features planned include JWT-based authentication for secure user login and registration, the ability for users to create, edit, and delete posts, real-time messaging, and Redis API for caching.

The application is being designed with scalability and user experience in mind, capable of handling a large number of users and posts while maintaining a user-friendly interface.

Technologies used

  • React.js
  • Node.js
  • Express.js
  • MongoDB
  • JWT Authentication
