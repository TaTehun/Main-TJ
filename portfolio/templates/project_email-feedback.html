The User-Email-Feedback application is an all-encompassing platform designed to simplify the often cumbersome process of gathering user feedback through emails.

Developed using the MERN stack, this application provides a seamless experience from the point of creating a survey to the final step of analyzing the feedback.

It features a secure OAuth-based authentication system, an intuitive survey creation interface, automated email distribution to recipients, and a real-time dashboard for feedback analysis.

The application is designed with scalability and user experience in mind, capable of handling a large volume of surveys and responses while maintaining a user-friendly interface.

Technologies used

  • React.js
  • Redux
  • Node.js
  • Express.js
  • MongoDB
  • OAuth
